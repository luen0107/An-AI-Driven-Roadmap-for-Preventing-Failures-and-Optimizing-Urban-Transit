# An-AI-Driven-Roadmap-for-Preventing-Failures-and-Optimizing-Urban-Transit

Enhancing LRT Station Ridership and Reliability: An AI-Driven Roadmap

This project focuses on leveraging artificial intelligence and data analytics to optimize urban transit systems, specifically targeting the Kelana Jaya LRT Line. By utilizing historical ridership and maintenance data from 2019 to 2024, the study aims to improve operational reliability and increase passenger satisfaction through predictive modeling.

Project Objectives

Analyze System Relationships: Identify the correlations between system failures, service downtime, and total ridership using Multivariate Regression Analysis.

Forecast Ridership: Predict future ridership patterns for the Ara Damansara Station, particularly following Transit-Oriented Development (TOD) projects, using the ARIMA model.

Optimize Maintenance: Develop data-driven maintenance schedules to minimize system downtime and improve overall reliability through Fault Tree Analysis (FTA).

Key Methodology

The project follows a comprehensive data science workflow:

1) Data Acquisition & Understanding: Processing confidential secondary data from Prasarana Malaysia, including monthly ridership and failure logs.

2) Data Preparation: Transforming raw datasets through cleaning, integration, and manual extraction to ensure high-quality inputs for modeling.

3) Advanced Modeling: Applying diverse analytical techniques including:

Time Series Analysis (ARIMA) for demand forecasting.

Multivariate Data Analysis to examine impact factors on service uptime.

Risk Analysis (FTA) to identify critical failure causes in system components.

Impact

By integrating AI-driven insights into urban transit, this project supports Malaysia's National Artificial Intelligence Roadmap (AI-RMap), aiming to boost productivity and competitiveness in the public transportation sector.
